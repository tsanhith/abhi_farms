# Abhi Farms

Pure Buffalo Milk delivered straight from our farm to your home.

## Project Overview

Abhi Farms is a family-owned dairy farm with three generations of experience. We provide 100% pure, farm-fresh buffalo milk and dairy products (Paneer, Ghee, Dahi) with a commitment to quality and zero preservatives.

## Features

- **Responsive Design**: Modern, premium UI that works on all devices.
- **WhatsApp Integration**: Easy ordering through direct WhatsApp links.
- **Product Showcase**: Detailed information about our dairy products.
- **Micro-animations**: Smooth scrolling and entrance animations.
- **Information Sections**: About Us, Quality Assurance, FAQ, and Contact pages.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- Google Fonts (Inter)

## How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/tsanhith/abhi_farms.git
   ```
2. Open `index.html` in your favorite web browser.
